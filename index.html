План колонізації Марсу

Бачення

Наша мета-відправити вантажну місію на Марс у 2022році.Ця місія повинна перевірити наявність води,визначити основні джерела небезпекита доставити вантажі,необхідні для розгортання первинної інфраструктури для генераціх енергії,видобутку корисних копалин та підтримки життя.

    План

  1. 2018:Побудувати допоміжну базу для запусків Бока Чіка
  2. 2019:Дебют ракети Falcon Big
  3. 2020-2021:Тестові запуски BFR і вивід її на орбіту
  4. 2022:Запуск двох місій до Марсу з вантажем,але без команди
